## 📋 Descrição do projeto
Este projeto foi desenvolvido como parte das atividades da **Agenda 12** da disciplina de **Desenvolvimento de Sistemas I** na Etec. O foco principal foi a aplicação de **HTML5 semântico** para a criação de um diretório organizado de sites favoritos.

O projeto utiliza estruturas modernas para garantir que o conteúdo seja acessível e bem interpretado pelos navegadores e motores de busca.

## 🚀 Funcionalidades
*   **Semântica HTML5**: Uso correto de tags como `<h1>`, `<h2>`, `<figure>` e `<figcaption>`.
*   **Organização Visual**: Lista de sites favoritos com logotipos e legendas padronizadas.
*   **Navegação Externa**: Links configurados para abertura em novas abas através do atributo `target="_blank"`.
*   **Portabilidade**: Caminhos relativos para garantir que as imagens funcionem em qualquer ambiente.

## 🛠️ Tecnologias e ferramentas
*   **Linguagem**: HTML5.
*   **Editor**: Visual Studio Code.
*   **Hospedagem**: GitHub.

## 📖 Como executar
1. Faça o download ou clone este repositório.
2. Certifique-se de que as imagens (`.png`) estão na mesma pasta que o arquivo `index.html`.
3. Abra o arquivo `index.html` em seu navegador de preferência.
